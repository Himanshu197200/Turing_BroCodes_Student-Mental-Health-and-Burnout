# Raw Dataset Link

Due to GitHub's strict 100MB file size limit, the raw `student_health_raw.csv` file (280.61 MB) cannot be hosted directly in this repository.

You can download the exact, pristine dataset here:
**[Kaggle: Student Mental Health Analysis Dataset](https://www.kaggle.com/datasets/ayeshasiddiqa123/student-health/data)**

## Local Setup Instructions:
1. Download the CSV file from the Kaggle link above.
2. Ensure the file is named exactly `student_health_raw.csv`.
3. Place it in this `data/raw/` directory.
4. **DO NOT** commit the CSV file to Git. Run the ETL pipeline script to begin analysis.
